# Pharmacy Billing & Inventory Management System

## Problem Statement
Pharmacies often face challenges in managing inventory, billing, and customer records efficiently. Manual processes can lead to errors, stockouts, and delays, impacting both business operations and customer satisfaction.

## Solution & Features
This project provides a comprehensive Pharmacy Billing & Inventory Management System with the following features:

- **User Authentication:** Secure login for staff.
- **Dashboard:** Overview of sales, inventory, and key metrics.
- **Inventory Management:** Add, update, and track medicines with stock alerts.
- **Billing System:** Generate and print bills for customers, manage cart, and apply discounts.
- **Customer Management:** Maintain customer records and purchase history.
- **Reports:** Generate sales and inventory reports.
- **Responsive UI:** Modern, mobile-friendly interface.

## Tech Stack
- **Frontend:** React (Vite), TypeScript, Tailwind CSS
- **Backend:** Java Spring Boot, Maven
- **Database:** MySQL
- **Other Tools:** Docker, Netlify/Vercel (deployment), PowerShell scripts for setup
